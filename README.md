# 🔥 Premium Emoji Bot

Convert any normal emoji to Telegram Premium Animated Emojis for FREE!

## 🚀 Features

- ✨ Convert any emoji to premium animated emoji
- 📝 Supports text, photo, video, and documents
- 🎨 Preserves all formatting (Bold, Italic, Links, etc.)
- 🔗 Add up to 4 inline buttons with colorful design
- 👥 Force join channel system
- 🔐 Admin panel with multiple admins

## 📋 Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/help` | Show help message |
| `/cancel` | Cancel current operation |
| `/admin` | Admin panel (Admins only) |

## 🎯 How to Use

1. Send `/start` to the bot
2. Tap **✍️ MAKE POST** button
3. Send your text message
4. Choose media (Video/Image/Document/Skip)
5. Choose number of buttons (1-4 or None)
6. Enter button names and links
7. Preview and click **✅ DONE**

## 👑 Admin Commands

| Command | Description |
|---------|-------------|
| 🔴 BOT OFF | Turn bot offline |
| 🟢 BOT ON | Turn bot online |
| 📢 BROADCAST | Send message to all users |
| 👥 USERS LIST | View all registered users |
| 📊 BOT STATS | View bot statistics |
